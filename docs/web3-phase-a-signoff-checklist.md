# Web3 Phase A Sign-off Checklist

Этот документ фиксирует обязательный sign-off перед инженерной фазой `B` (деплой минимального `BET` ERC-20 в тестнете).

## 1) Product and token economics

- Utility токена `BET` и ограничения первого релиза согласованы.
- `initialSupply`, роли минтинга и burn-политика утверждены.
- Перечень операций, доступных до mainnet (sandbox-only), зафиксирован.

## 2) Security and operations

- Определены владельцы `admin`/`pauser` ролей и ротация ключей.
- Согласован emergency-process (pause/unpause, отзыв ролей).
- Определён порог инцидента, при котором тестнет деплой блокируется.

## 3) Network choice

- Выбрана целевая тестовая сеть: `________________`.
- Подтверждены RPC-провайдеры и резервный endpoint.
- Согласованы ограничения faucet/тестовых токенов для CI/CD.

## 4) Delivery gates

- Готовы артефакты ABI и адреса контрактов для индексатора (Phase C).
- Определён формат конфигурации для backend (`.env`/Secret/TF vars).
- Назначены ответственные за smoke-проверки после деплоя.

## 5) Sign-off record

- Дата: `YYYY-MM-DD`
- Product: `________________`
- Backend: `________________`
- Security: `________________`
- DevOps: `________________`

Если любой пункт выше не закрыт, Phase B не стартует.
